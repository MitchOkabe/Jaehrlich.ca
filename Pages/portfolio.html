Portfolio


Jaehrlich.ca - Portfolio Website

Active: 2020 - Now

Languages used: AMP HTML, CSS


Building an ultra fast, moble-first AMP website to display my portfolio and technical blog.


With the majority of website visitors coming from mobile devices, and load times turning away visitors, web developers can use AMP HTML to design for fast results. This site shows how to create an extremely light weight and responsive site with minimal javascript.

Developed using Visual Studio Code and GitHub to push changes to the live site automatically, for rapid development. Modern templating solutions, like bootstrap, are not supported in an AMP HTML site, so basic HTML and CSS is used to create a simple and effective style.

AutoTask Desktop - IT Management

Active: 2019 - Now

Languages used: C#, XAML, SQL, XML

Windows Presentation Format


Building the IT platform used for managing tickets and client information at Resilient IT.


AutoTask is a web-based ticketing platforms for IT service providers. The AutoTask Desktop application brings all their web features to the desktop, along with invaluable time saving workflows and features for employees of Resilient IT.

Built using XAML and C# on the Windows Presentation Format, this app runs fast on any Windows computer of any screen size. Multithreading design practices enable instant reponse from user action, while delivering large amounts of data. Integration with Microsoft Outlook allows for easy response via email.



Ticket Board - IT Info

Active: 2016 - Now

Languages used: C#, XAML, SQL, XML

Windows Presentation Format


Building a large dashboard to see all active and incoming tickets at a glance


The Ticket Board application displays a visual representation of tickets on a large screen TV in the office. This app allows employees of Resilient IT to move freely thoughout the office while maintaining knowledge of the current ticket climate.

This app also doubles as a ticket parsing server, automatically categorizing tickets, attaching contact information, and notifying out-of-office employees of new tickets via email.
